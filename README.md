# Cascade

For now, Cascade defines a minimal and automated path to booting a [stage 3 Gentoo tarball](https://wiki.gentoo.org/wiki/Stage_file#Stage_3).

## Run Cascade in a QEMU virtual machine

```sh
just vm
```
