# Cascade

For now, Cascade defines a minimal and automated path to replace GNU coreutils with uutils coreutils in a [stage 3 Gentoo tarball](https://wiki.gentoo.org/wiki/Stage_file#Stage_3).

```sh
just clean chroot
```
